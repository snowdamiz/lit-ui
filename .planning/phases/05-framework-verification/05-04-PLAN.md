<task type="checkpoint:human-verify" gate="blocking">
  <what-built>Three framework test applications (React 19, Vue 3, Svelte 5) each demonstrating lit-ui Button and Dialog components</what-built>
  <how-to-verify>
    For EACH framework app (React, Vue, Svelte):

    **Button Component:**
    1. Verify all 5 variants render (primary, secondary, outline, ghost, destructive)
    2. Verify all 3 sizes render (sm, md, lg)
    3. Click a button - console should log "Button clicked"
    4. Verify disabled button shows disabled state
    5. Click loading button - should show spinner for 2 seconds
    6. Verify icon slots render if present

    **Dialog Component:**
    7. Click "Open Dialog" button - dialog should appear with animation
    8. Verify title slot renders in dialog header
    9. Verify content renders in dialog body
    10. Verify footer slot renders with close button
    11. Press Escape - dialog closes, console shows "close" with reason "escape"
    12. Reopen and click backdrop - dialog closes with reason "backdrop"
    13. Reopen and click close button - dialog closes with reason "programmatic"

    **Form Participation:**
    14. Find form with submit button
    15. Click submit - form should submit (may show in network tab or form behavior)

    **No Errors:**
    16. Open DevTools Console - verify NO errors or warnings
    17. For Vue specifically: verify no "failed to resolve component" warnings

    URLs:
    - React: http://localhost:5173
    - Vue: http://localhost:5174
    - Svelte: http://localhost:5175
  </how-to-verify>
