From scripts/compile-knowledge.ts (Phase 106 — xmlEscape defines exact 4 entities to reverse):
```typescript
// xmlEscape encodes in this order: & → &amp;, < → &lt;, > → &gt;, " → &quot;
// decodeEntities MUST reverse in opposite order (&amp; LAST — prevents double-decode):
function decodeEntities(text: string): string {
  return text
    .replace(/&quot;/g, '"')
    .replace(/&gt;/g, '>')
    .replace(/&lt;/g, '<')
    .replace(/&amp;/g, '&')   // MUST be last
}

// </skills>
//
// Section content lines have 2 leading spaces from:
```
