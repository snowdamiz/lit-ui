`switchProps: PropDef[]` — 7 props:

- required (boolean, false) — Whether the switch must be on for form submission
- name (string, "") — Form submission name
- value (string, "on") — Form submission value when checked
- label (string, "") — Label text displayed next to the switch
- size ("sm" | "md" | "lg", "md") — Size variant

4. `disabledCode` — Disabled states (off + on)
5. `requiredCode` — Required switch for terms acceptance
6. `formCode` — Switch in a form showing form participation (name, value, submission)
7. `multipleCode` — Multiple switches for settings panel pattern (notifications, dark mode, auto-save)
