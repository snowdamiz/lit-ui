CRITICAL — Chart data binding pattern (NOT HTML attribute serialization):
Charts require .data property assignment (not HTML attribute). Use this React pattern for every demo:
```tsx
import { useEffect, useRef } from 'react';

function LineChartDemo() {
  const ref = useRef<HTMLElement>(null);
  useEffect(() => {
    if (ref.current) {
      (ref.current as any).data = [
        { name: 'Sales', data: [120, 200, 150, 80, 70, 110, 130] },
        { name: 'Revenue', data: [60, 100, 80, 40, 50, 75, 90] },
      ];
    }
  }, []);
  return <lui-line-chart ref={ref as any} smooth zoom style={{ height: '300px', display: 'block' }} />;
}
```
This pattern is REQUIRED for all 4 demo functions in this plan.

- data: chart-specific type | undefined — Chart data (JS property only)
- option: EChartsOption | undefined — Raw ECharts option passthrough

- max-points: number, default 1000 — Streaming buffer capacity

LuiLineChart specific: smooth:boolean(false), zoom:boolean(false), mark-lines:MarkLineSpec[](undefined)

LuiPieChart specific: min-percent:number(0), inner-radius:string|number(''), center-label:string('')
