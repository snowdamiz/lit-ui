## Architecture Consistency

All 3 component packages follow identical patterns:

- **Package structure:** package.json, tsconfig.json, vite.config.ts, src/index.ts, src/jsx.d.ts
- **Base class:** TailwindElement from @lit-ui/core
- **Theming:** CSS custom properties (--ui-{component}-*)
- **Form participation:** ElementInternals (Switch, Checkbox, RadioGroup own form; CheckboxGroup and Radio delegate)
- **Registration:** Safe customElements.define with collision detection + SSR guard
- **JSX types:** React, Vue, Svelte declarations
